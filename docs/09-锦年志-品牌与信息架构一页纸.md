# 锦年志：品牌与信息架构一页纸

## 1. 网站定位
锦年志是一个以“时光日志”为核心叙事的对外博客站点，兼顾个人表达与开放共创。首期目标不是商业化转化，而是持续输出有温度的内容与可参与的创作空间。

## 2. 核心受众
- 18-35 岁、习惯数字写作和内容消费的青年用户。
- 对记录成长、知识沉淀、生活观察有持续表达需求的创作者。
- 对开放讨论、评论互动、共同创作有兴趣的读者与贡献者。

## 3. 品牌主题与关键词
- 品牌名：锦年志
- 语义锚点：锦绣年华、风华正茂、时光日志
- 情绪关键词：青春活力、温暖人文、真实记录、共同生长

## 4. 内容策略优先级
1. 首页：展示精选日志与品牌主张，承担“第一印象 + 共创召集”职责。
2. 探索：承载标签驱动的发现体验，满足“无预设分类、自然增长”的内容组织方式。
3. 博客详情：承载评论互动，形成内容讨论闭环。
4. 关于与联系：建立作者人格与外部连接。

## 5. 首版信息架构
- 顶部导航：`首页 / 探索 / 关于 / 搜索 / 登录(加入我们)`
- 首页：品牌引导区 + 日志流 + 共创入口
- 探索页：左侧“堆积式标签侧栏” + 右侧内容区
- 页脚：品牌说明、联系与社媒、隐私与条款

## 6. 交互边界
- 共创入口：在首页和登录后后台双入口并行。
- 评论入口：仅在日志详情页内出现，不在首页和导航中直接暴露。
- 搜索入口：始终在导航栏，作为全局能力。

## 7. 视觉语言原则
- 气质：温暖人文为底色，青春活力为强调。
- 版式：清晰留白，减少装饰噪音，突出内容本身。
- 风格：延续现有玻璃态与柔和阴影体系，避免过冷科技感或重商务感。
