# 关于实例


### 功能一览

支持的功能列表：
- 租用
- 退租
- 续租
- 自动续租

资源管理：
- 重启实例
- 重置系统

资源升级：
- 可增加内存配置，不可降配
- 可增加CPU配置，不可降配
- 可添加数据盘，不可删除
- 系统盘无法更改大小
- 数据盘可以添加最多16块

访问开放：
- 可通过外部ssh登录；可通过web端ssh登录
- 可通过vnc登录；
- 可通过rdp登录；
- 无独立ip，有10个端口



### 关于实例费用字段

[实例详情接口](https://apifox.com/apidoc/shared-b0fc397f-c455-4c9a-9d82-875fc48ae106/api-240550511) 有如下字段：

- Total_cost，实例基础费用（包含初始GPU+CPU+内存、添加CPU、添加内存，续费等）
- BandwidthCost，带宽费用：收费带宽大小 * 带宽单价 * 小时时长
- AddDiskCost，添加磁盘的费用 * 磁盘单价 * 此刻到实例到期的小时时长
- KeeppedDiskCost，保留磁盘费用：（系统盘大小+数据盘大小）* 磁盘单价 * 小时时长
- ReturnMoney，已退费的金额

最终实例已产生的总费用 = Total_cost + BandwidthCost + AddDiskCost + KeeppedDiskCost

### 关于实例时间字段

[实例详情接口](https://apifox.com/apidoc/shared-b0fc397f-c455-4c9a-9d82-875fc48ae106/api-240550511) 有如下字段：

- Ctime，创建时间
- Due_time，到期时间
- End_time，实际结束时间
- DiskReleaseTime，磁盘释放时间
- LastRestartTime，上一次重启时间(可为空)
- LastAbnoramlTime，上一次发生异常时间(可为空)
- AbornormalTime，异常时长(秒)

### 关于实例资源相关字段

[实例详情接口](https://apifox.com/apidoc/shared-b0fc397f-c455-4c9a-9d82-875fc48ae106/api-240550511) 有如下字段：

- Cpu_num、Memory、Gpu_num，为初始状态下的资源大小；其中系统盘大小字段没有，固定为(linux 200GB，windows 350GB)
- AddCpu、AddMemory、AddDisk，为已经添加的资源大小；
- AddingCpu、AddingMemory、AddingDisk：为正在添加的资源大小（状态未完成的话不扣费）；
- Bandwidth、MaxBandwidth、NoFreeBandwidth：为带宽相关，分别是当前带宽大小、可设置的最大带宽、收费带宽大小；每小时扣费
