# 开始使用


### 准备工作

1.注册用户：前往 [智星云|算力官网](https://gpu.ai-galaxy.cn/store)，完成注册。
2.实名认证：使用OpenAPI要求完成实名认证（个人实名、企业认证都可以），入口 [智星云|实名认证](https://gpu.ai-galaxy.cn/console/verification/selectVerification)。
3.获取AccessKey：从官网进入【控制台 -> 开放API -> AccessKey管理】页面，点击创建按钮，填写对应信息完成创建，入口 [智星云|OpenAPI|AccessKey管理](https://gpu.ai-galaxy.cn/console/openapi/accessKey)。


### 信息准备

- 接口调用前缀：`https://app.ai-galaxy.cn`
- apikey：上一节获取的AccessKey
- apisecret：上一节获取的SecretKey
