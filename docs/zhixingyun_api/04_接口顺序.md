# 接口顺序


### 租用市场

- 获取基础配置项 /openapi/v2/store/get_configs
- 获取价格信息 /openapi/v2/store/get_price_info

租用方式1：老版
- 获取云主机租用选项 /openapi/v2/store/get_kvm_available_selector
- 租用:创建云主机 /openapi/v2/store/create_kvm_instance

租用方式2：新版
- 获取实例规格列表 /openapi/v2/store/get_instance_specifications
- 租用:创建云主机 /openapi/v2/store/create_kvm_instance

### 实例管理

- 获取实例统计 /openapi/v2/instance/get_instance_status_count
- 获取列表 /openapi/v2/instance/get_instance_list
    - 获取详情 /openapi/v2/instance/get_instance_detail
        - 添加磁盘 /openapi/v2/instance/add_disk
        - 添加CPU /openapi/v2/instance/add_cpu
        - 添加内存 /openapi/v2/instance/add_memory
        - 修改带宽 /openapi/v2/instance/change_bandwidth
    - 获取实例费用
        - 费用汇总信息 /openapi/v2/billing/get_instance_cost_summary
        - 费用明细列表 /openapi/v2/billing/get_instance_cost_list
    - 保留磁盘状态
        - 释放磁盘 /openapi/v2/instance/release_keepped_disk
        - 从保留磁盘启动 /openapi/v2/instance/create_instance_from_keepped_disk
    - 重置系统
        - 获取重置可用的镜像 /openapi/v2/instance/get_local_images
        - 重置系统 /openapi/v2/instance/reinit_system
    - 续费管理
        - 手动续费 /openapi/v2/instance/renew_instance
        - 自动续费 /openapi/v2/instance/set_autorenew
        - 实例启动错误，手动退费 /openapi/v2/instance/error_refund
    - 实例信息管理
        - 修改密码 /openapi/v2/instance/change_password
        - 修改备注 /openapi/v2/instance/change_note
        - 修改实例扣费优先级 /openapi/v2/instance/change_pay_type_first
        - 修改实例到期处理模式 /openapi/v2/instance/change_due_mode
    - 重启实例 /openapi/v2/instance/restart_instance

### 账单管理

- 获取充值列表 /openapi/v2/billing/get_recharge_order_list
- 获取账户变更明细 /openapi/v2/billing/get_balance_change_list

### 账户管理

- 获取主账户信息 /openapi/v2/account/get_main_account_info
