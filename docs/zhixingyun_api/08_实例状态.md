# 实例状态


### 实例状态

按照场景介绍状态流转：

**租用场景**：租用后，Status==4，表示启动中
Status==1，表示运行中

**启动异常场景**：
启动超时（启动异常）后：Status==-1
Status==-1的状态，用户可以点击异常退费
异常退费后，Status==-2，表示异常已退费

**点击重启场景**：Status==5，表示重启中
重启成功后，Status==1，表示运行中
重启失败，Status==7，比较少出现，需要运维人员介入

**租用实例到期后**：Status==0，表示已结束，已释放
如果用户选择了保留磁盘，实例到期后，Status==8
Status==8的状态，用户可以点击从保留磁盘重启
用户点击从保留磁盘重启后，会重新创建新的实例，状态从Status==4开始


### 逻辑分组
智星云控制台页面上的逻辑分组：

默认(statusDefault)：        Status=1,4,5,-1,7,8
运行中(statusRunning)：       Status=1,4,5
保留磁盘(statusKeeppedDisk)： Status=8
运行异常(statusRunningError)：Status=1,4,5 && IsAbnormal!=0
启动错误(statusCreateError)： Status=-1,-2,7
全部(statusAll)：            不做过滤，全部返回
