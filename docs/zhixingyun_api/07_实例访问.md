# 实例访问

### 镜像为Windows系统

#### 1. windows远程连接
远程桌面地址：字段`Url`+`RdpPort`
用户名：固定为 `vipuser`
密码：字段为 `RdpPasswd`

#### 2. VNC
VNC地址：字段`Url`
端口：字段`VncPort`
用户名：固定为 `vipuser`
密码：字段为 `VncPasswd`

#### 3. 预留端口
外网开放端口，字段为`ExtraPort`，名称依次排序：p1,p2,p3,p4,p5,p6
对应内网字段为：`PortMappingInner`，一一对应p1,p2,p3,p4,p5,p6


### 镜像为Linux系统

#### 1. SSH连接
远程桌面地址：字段`Url`+`SshPort`
用户名：固定为 `root`
密码：字段为 `Init_passwd`
SSH登录命令：ssh root@`Url` -p `SshPort`

#### 2. VNC
VNC地址：字段`Url`
端口：字段`VncPort`
用户名：固定为 `root`
密码：字段为 `VncPasswd`

#### 3. 预留端口
外网开放端口，字段为`ExtraPort`，名称依次排序：p1,p2,p3,p4,p5,p6
对应内网字段为：`PortMappingInner`，一一对应p1,p2,p3,p4,p5,p6
